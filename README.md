# 🚀 Mastering Modern JavaScript

Welcome to my Modern JavaScript learning repository! I created this space to track my progress, store my notes, and showcase the mini-projects I build as I dive deep into modern JS (ES6 and beyond).

## 🎯 Goals for this Repository
*   **Build a strong foundation:** Master core concepts, scope, and closures.
*   **Embrace Modern Syntax:** Get comfortable with ES6+ features (arrow functions, destructuring, spread/rest operators).
*   **Master Asynchronous JS:** Understand Promises, `async/await`, and API fetching.
*   **Create Real Projects:** Apply what I learn to build functional, interactive web components.
